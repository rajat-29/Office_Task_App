# Office_Task_App
A Simple Task App with update and delete features and firebase backend!!

## Feature
<ul>
<li>Easy to Use</li>
<li>Update Notes</li>
<li>Delete Notes</li>
<li>Recycler View</li>
<li>User Friendly Interface</li>
<li>Login/Register Functionality</li>
</ul>
